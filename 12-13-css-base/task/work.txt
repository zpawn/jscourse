Описать html, css для расположения основных блоков страницы (index_areas.jpg). Конкретное содержимое блоков можно не детализировать. Блоки подсветить разными цветами. Разметка областей в файле index_areas.jpg условная, это значит, что можно создавать блоки с отступами, или иначе группировать блоки. Теме не менее результат верстки должен быть похожим на исходный макет.
Ширины блоков должны быть процентными (не фиксированными в 400px итд). Верстка должна позволять блокам размещать в себе разное количество контента (значит нельзя фиксировать высоту блоков).
